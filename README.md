## 1 Global Sales Analysis

**The Problem:** A global retail business had 113,036 sales records with an incorrect revenue formula affecting all transactions, making performance tracking unreliable

**The Tools Used:** Microsoft Excel · Pivot Tables · Power Query · Slicers

**The Impact:** Corrected the revenue formula, cleaned 1,000 duplicate records, and built a two-page interactive dashboard revealing that Bikes generated 69% of revenue but Accessories delivered superior 47% profit margins - flagged as a key growth opportunity

## 2 DePaul University - Outreach and Application Performance Analysis

**The Problem:** The admissions team had 7,544 disorganised raw records with a data quality score of just 38.4%, making it impossible to track application pipeline performance or identify bottlenecks

**The Tools Used:** Microsoft Excel · Power BI · DAX · Power Query

**The Impact:** Cleaned data to 4,999 quality records, built an interactive dashboard, and identified that transcript issues were stalling 580+ applications, the single largest pipeline blocker — while flagging a 61% geographic concentration risk from India

## 4  Data Professional Survey Dashboard

**The Problem:** HR teams and industry leaders lacked a clear picture of what data professionals earn, what skills they use, and why so many are considering leaving their roles

**The Tools Used**: Power BI · Power Query · DAX

**The Impact:** Analysed 369 survey responses and revealed that 40.67% of data professionals intend to switch careers, a critical workforce retention signal while confirming Python and SQL as the most in-demand skills across all industries
